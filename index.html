SKILLS

HTML 5

HTML, è il linguaggio standard utilizzato per la creazione e la strutturazione delle pagine web. fornendo così le fondamenta su cui si basa il web moderno.

CSS 3

CSS, è un linguaggio utilizzato per definire l'aspetto e la formattazione di documenti HTML e XML. Permette di personalizzare l'aspetto di una pagina web, consentendo agli sviluppatori di creare design accattivanti.

PHP

PHP è un linguaggio di scripting per lo sviluppo di applicazioni web dinamiche. Consente di creare siti web con funzionalità avanzate, contribuendo così a fornire esperienze web dinamiche e personalizzate.

Javascript

JavaScript è un linguaggio di programmazione utilizzato per rendere le pagine web interattive. Consente di aggiungere animazioni, validazione dei moduli e aggiornamenti in tempo reale.

Angular

Angular è un framework sviluppato da Google, utilizzato per la creazione di applicazioni web e mobile moderne e scalabili. Semplifica lo sviluppo di interfacce complesse e offre strumenti avanzati.

React

React è una libreria JavaScript sviluppata da Facebook, Grazie al suo approccio basato sui componenti e alla gestione efficiente dello stato, offre una soluzione potente per lo sviluppo di UI moderne e performanti.
